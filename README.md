# Radon Series Fit By Flag
Это одностраничное приложение для обработки временных серий объемной активности (ОА) радона, полученных с радон-мониторов.
Доступно по ссылке Radon [Series Fit By Flag](https://vsz2020.github.io/RadonSeriesFitByFlag/).

Приложение дает возможность удобной обработки участков накопления радона в помещении и участков разбавления (за счет притока уличного воздуха) радона. 
Пользователь лишь выбирает участки временной серии, которые будут обработаны приложением автоматически.
Выбранные участки серии подгоняются под функцию вида

C(t)=C0 exp(-lambda t) + phi/lambda exp(-lambda t)

Для подгонки выбранного участка под модельную функцию накопления/разбавления ОА радона применяется стандартный метод Левенберга-Марквардта.

## Как работать с программой
1. Импортировать временную серию (из буфера обмена, файла TXT/CSV, вставкой в поле ввода).
2. На графике отобразится временная серия.
3. Нажать кнопку "Флаг 1" или "Флаг 0" для дальнейшего выбора участка накопления или разбавления ОА радона.
4. Масштабировать нужный участок временной серии вращением колесика мыши.
5. Зажав Ctrl, выделить левой кнопкой мыши (ЛКМ) интервал, значениям которого будет присвоен "Флаг 1".
6. Очистить присвоенные флаги можно зажав Ctrl+ПКМ (правую кнопку мыши) и выбрав интервал для очистки.
7. Повторить для других интервалов и другого флага (Флаг 0).
8. Программа автоматически проведет подгонку каждого интервала и выведет оптимальные значения параметров phi, lambda, C0.
9. Все данные по интервалам с оптимальными параметрами можно экспортировать в файл TXT/CSV или скопировать в буфер обмена.
