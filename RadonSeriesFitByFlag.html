Обработка временной серии радона с подгонкой по Левенбергу-Марквардту (LMA)

Исходная серия
Поддерживаются разделители таб, запятая и точка с запятой. Если есть третья колонка, она может содержать флаг 0/1.
или
Как работать:
1) загрузите временную серию в поле для ввода и нажмите «Загрузить из поля для ввода»;
2) выберите режим «Флаг 1», «Флаг 0»;
3) удерживайте Ctrl и выделяйте интервал точек, которым нужно установить флаг;
4) по окончании работы экспортируйте таблицу со всеми данными.
Статус: данные не загружены Курсор: - Fit: - Выделение: -
Выберите флаг | |
Ctrl + ЛКМ с выделением области - назначить флаг Ctrl + ПКМ с выделением области - снять флаг Колесо - масштаб по X вокруг курсора Shift + Колесо - масштаб по Y вокруг курсора ЛКМ - панорамирование
